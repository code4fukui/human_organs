# Human Organs

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Human Organs is a web application that visualizes and shares 3D human organ data, utilizing data from the NIH 3D Print Exchange.

## Features

- 3D data (GLB, STL) of various human organs
- Images and metadata for the 3D models
- Data sourced from the NIH 3D Print Exchange

## Data / API

The 3D data and metadata are sourced from the [NIH 3D Print Exchange](https://3d.nih.gov/).
