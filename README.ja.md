# Human Organs

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Human Organsは、NIH 3D Print Exchangeからのデータを活用し、3Dの人間の臓器データを可視化および共有するウェブアプリケーションです。

## 特徴

- さまざまな人体の臓器の3Dデータ (GLB, STL)
- 3Dモデルの画像とメタデータ
- NIH 3D Print Exchange から提供されたデータ

## データ / API

3Dデータおよびメタデータは、[NIH 3D Print Exchange](https://3d.nih.gov/) から取得されています。
